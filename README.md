- papaparse, eslint, prettier

## Installation

App requires [Node.js](https://nodejs.org/) v14+ to run.

Install the dependencies and start the project.

```sh
cd project
npm install
```

#

## How to use

### commands

1- run the app

```sh
npm run start
```

2- run unit test

```sh
npm run test
```

3- run e2e test

```sh
npm run cypress:open
```

4- run lint and fix error

```sh
npm run lint
npm run lint:fix
```

5- run format

```sh
npm run format
```

6- for production

```sh
npm run build
```
